# Jogo da Forca em Python

Projeto desenvolvido como atividade acadêmica utilizando Python.

## Sobre o projeto

O programa implementa o clássico jogo da forca em modo texto, permitindo ao usuário adivinhar uma palavra através de tentativas limitadas.

## Conceitos utilizados

- Variáveis
- Condicionais (if/else)
- Laços de repetição (while)
- Manipulação de strings
- Listas
- Entrada e saída de dados

## Como executar

1. Baixe o arquivo `jogo_da_forca.py`
2. Execute o programa em um ambiente Python
3. Siga as instruções exibidas no terminal
